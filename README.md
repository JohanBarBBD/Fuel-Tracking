## Docker Create local msswl Database and run the creation script
- 1. Open a terminal in the Fuel-Tracking directory 
- 2. enter this into the terminal "cd docker" 
- 3. enter this into the terminal docker compose up -d
- 4. wait 30 seconds and your local db will be up and running
